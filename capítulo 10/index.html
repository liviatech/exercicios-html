hreflan indica o idioma principal do site para onde o link está desviando o fluxo de navegação Site w3schools

Para controlar o site destino podemos usar o atributo target com _self continuar navegando no site

target _top abrir em um navegador completo

target _blank abrir em uma nova janela

target _parent similar ao uso do top em uma referência a janela mãe

target nome-do-frame caso esteja usando frame indicar o nome do frame que abrir

Vamos nos basear em _blank e _self

Indicar qual a natureza do destino usando o atributo rel, esse atributo aceita vários valores

rel next indica que o link é para a próxima parte do documento atual Continuar navegando no site

prev indica que o link é para a parte anterior do documento atual

author indica que é umlink para o site do autor do artigo atual

external indica que é um link para outro site que não faz parte do site

No código anterior o primeiro link chamamos de ink local ou link interno já que leva o visitante outra página do nosso próprio site, já o segunndo link chamamos de link externo ai devemos indicar o link completo
